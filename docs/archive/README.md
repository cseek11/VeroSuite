# Documentation Archive

This directory contains historical documentation that has been completed or superseded by newer documentation.

## Archive Structure

### Inconsistency Reports
- `inconsistency-reports/` - Completed inconsistency analyses and reports
  - These reports documented codebase inconsistencies that have been addressed
  - Reference only - do not use for active development

### Implementation Summaries
- `implementation-summaries/` - Feature completion summaries and status reports
  - Historical records of completed features and fixes
  - Useful for understanding project history

### Migration Notes
- `migration-notes/` - Completed migration documentation
  - Notes from completed migrations
  - Reference for understanding past changes

### Developer Tickets
- `developer-tickets/` - Completed developer tickets
  - Historical tickets for reference
  - Active tickets remain in `DEVELOPER_TICKETS/`

## Archive Policy

- Documentation is moved to archive after 60 days of inactivity
- Archived docs are marked with `status: archived` in frontmatter
- Archived docs are not included in active documentation navigation
- Archived docs are preserved for historical reference

## Finding Archived Documentation

To find archived documentation:
1. Check the appropriate subdirectory
2. Search by feature name or date
3. Review file names for context
