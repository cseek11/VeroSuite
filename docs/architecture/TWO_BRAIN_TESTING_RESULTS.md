   ```bash
   cat .cursor/llm_input.txt
   ```

2. **Apply fixes manually (or via Cursor):**
   - Read `.cursor/llm_input.txt`
   - Apply fixes from `ENFORCER_REPORT.json`
   - Write response to `.cursor/llm_output.txt`
   - Include `[FIX_COMPLETE]` tag

3. **Fix loop will continue:**
   - Detects response in `.cursor/llm_output.txt`
   - Re-runs enforcer to verify fixes
   - Loops until clean or max iterations
