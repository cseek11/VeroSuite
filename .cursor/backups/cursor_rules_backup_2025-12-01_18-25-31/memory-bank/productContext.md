# Product Context

**Last Updated:** 2025-11-30

## Why This Project Exists

VeroField is a field service management platform designed to help businesses manage field operations, work orders, customers, and technicians efficiently and securely.

## Problems Solved

### Multi-Tenant Field Service Management
- Secure tenant isolation for multiple organizations
- Comprehensive work order management
- Customer relationship management
- Technician scheduling and dispatch

### Security & Compliance
- Row Level Security (RLS) for data isolation
- Comprehensive audit logging
- SOC2/privacy compliance
- Secure authentication and authorization

### Code Quality & Consistency
- AI governance system for consistent code generation
- Pattern enforcement to prevent architectural drift
- Comprehensive testing and quality gates
- Documentation-driven development

## User Experience Goals

### For End Users
- Intuitive interface for managing work orders
- Efficient customer and technician management
- Real-time updates and notifications
- Mobile-first experience for field technicians

### For Developers
- Clear patterns and guidelines
- Comprehensive documentation
- AI-assisted development with governance
- Consistent code quality across monorepo

## Product Vision

A secure, scalable, and maintainable field service management platform that leverages AI assistance while maintaining strict quality and security standards.
